# PhatTea
 Phat tea is a type of iced tea developed to be distinct and powerful. The idea behind Phat Tea was to make a type of iced tea that doesn't cut corners, and is made from natural and easily accessible ingredients. The result is an iced tea that has a powerful, yet complex flavor, that can be made in a matter of minutes. _Looking for the recipes? On desktop, they should be under the code tab. On mobile, you have to tap "view code"._
